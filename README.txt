The easiest way to get valuable Mailgun metrics into Groundhogg!

== Description ==

This Mailgun Extension for Groundhogg makes tracking your mail a breeze. Directly integrated with Groundhogg, this extension allows you to:

* Track emails being opened
* Track delivery of emails
* Monitor link clicks in emails
* Update user location & timezone
* Automatically unsubscribe bogus or bouncing emails

As this extension gives you new and powerful benchmarks, you can clean, sort and order your lists in a seconds.

== Installation ==

We recommend viewing our documentation on WooNinja.io for full installation steps.

Installation is a breeze but remember to read all the steps, including how to set up Mailgun itself to report metrics to your site.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Groundhogg->Settings->Mailgun screen to configure the plugin
4. Enter your license key, click Save Changes, then click Activate.

== Frequently Asked Questions ==

= Why do I need this plugin? =

Mailgun provides amazing stats for your mail, including opens, deliveries and link clicks. Using our plugin, you can sort and clean your lists, in addition to knowing more about your user including location and timezone.

= What benchmarks are included in this plugin? =

This plugin includes the abilty to track delivery's, opens and clicks. In addition, the plugin can automatically update a users location and timezone (When provided by Mailgun) Lastly, there is an option auto-unsubscribe for permanent fail or bounce emails reported by Mailgun.

= Does this extension send mail via Mailgun? =

No, as there are many amazing plugins for sending mail via Mailgun so we decided to build a plugin to bring those amazing metrics back into Groundhogg.
